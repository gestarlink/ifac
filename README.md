# IFAC — Gestão de Projetos Esportivos e Educacionais

Plataforma desenvolvida pela **GestarLink** para o Instituto Formando Águias e Campeões (FAC).

## Sobre o projeto

Sistema completo de gestão administrativa para controle de alunos, inscrições, frequência, documentos, relatórios e projetos esportivos e educacionais.

## Funcionalidades

- Gestão de alunos e turmas
- Controle de inscrições e matrículas
- Registro de frequência e presença
- Emissão de documentos e certificados
- Relatórios e dashboards gerenciais
- Gestão financeira e de projetos

## Tecnologias

- React + TypeScript
- Vite
- Supabase
- Tailwind CSS
- Shadcn/ui
